# Making A Large Island
## Hard
<div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">You are given an<strong> n x n binary matrix </strong>grid. You are allowed to change <strong>at most one 0</strong> to be 1.<br></span><span style="font-size: 14pt;">Return the size of the <strong>largest island</strong> in grid after applying this operation.<br></span><span style="font-size: 14pt;">A group of connected 1s forms an island. <pre><span style="font-size: 14pt;"><strong><span style="font-size: 14pt;">Input:&nbsp;</span></strong><br><span style="font-size: 14pt;">[[1,0],[0,1]]</span><br><strong><span style="font-size: 14pt;">Output:</span></strong><br><span style="font-size: 14pt;">3</span><br><strong><span style="font-size: 14pt;">Explanation:</span></strong><br><span style="font-size: 18.6667px;">Change one 0 to 1 and connect two 1s, then we get an island with area = 3.</span><br></span></pre>
<p><strong><span style="font-size: 14pt;">Example 2:</span></strong></p>
<pre><strong><span style="font-size: 14pt;">Input:&nbsp;</span></strong><br><span style="font-size: 14pt;">[[1,1],[1,0]]</span><br><strong><span style="font-size: 14pt;">Output:</span></strong><br><span style="font-size: 14pt;">4</span><br><strong><span style="font-size: 14pt;">Explanation:</span></strong><br><span style="font-size: 18.6667px;">Change the 0 to 1 and make the island bigger, only one island with area = 4.<br></span></pre>

<pre><strong><span style="font-size: 14pt;">Input:&nbsp;</span></strong><br><span style="font-size: 14pt;">[[1,1],[1,1]]</span><br><strong><span style="font-size: 14pt;">Output:</span></strong><br><span style="font-size: 14pt;">4</span><br><strong><span style="font-size: 14pt;">Explanation:</span></strong><br><span style="font-size: 18.6667px;">Can't change any 0 to 1, only one island with area = 4.<br></span></pre>

<p><span style="font-size: 18.6667px;">You don't have to read input or print anything.
